# Harmonizing Web App

This package contains the Next.js application for Harmonizing.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run typecheck`
- `npm run prisma:generate`
- `npm run prisma:migrate`
- `npm run prisma:seed`

## Local env
See root `.env.example` and `apps/web/.env.example`.
