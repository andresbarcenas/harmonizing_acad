# UX System

## Brand Direction
- Premium minimalist visual tone
- Black/white foundation with warm soft-gold accent
- High whitespace density and rounded 2xl surfaces
- Calm motion and restrained shadows

## Tokens
- `--color-paper`, `--color-ink`, `--color-gold`, `--color-border`, `--color-muted`
- Typography:
  - Display: `Playfair Display`
  - Body/UI: `Manrope`

## Role Experience
- Student: reassuring membership-style cards and progress context
- Teacher: dense, scan-friendly operations and approval actions
- Admin: analytics + assignment controls in balanced dashboard cards

## Mobile-First Rules
- Header nav uses horizontal chips for compact touch use
- Core cards stack at small widths and split into grid on larger breakpoints
- High-contrast buttons and form controls for quick action flows
